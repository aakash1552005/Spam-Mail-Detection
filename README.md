> A machine learning model that classifies emails as Spam or Not Spam using Natural Language Processing (NLP) techniques.

---

## 📌 Overview

This project builds a text classification model to detect spam emails. It uses NLP techniques to process raw email text and trains ML models to accurately identify spam messages.

---

## ✨ Features

- ✅ Binary classification: Spam / Not Spam
- ✅ Text preprocessing & cleaning
- ✅ TF-IDF Vectorization
- ✅ Model training & evaluation
- ✅ Accuracy, Precision & Recall metrics

---

| Tool | Purpose |

| Google Colab | Development environment |

---

## 📂 Dataset

- **Source:** SMS Spam Collection Dataset (Kaggle)
- **Records:** 5,572 rows × 2 columns
- **Target:** ham = Not Spam, spam = Spam
- **Features:** Raw email/SMS text messages

---

## 📊 ML Models Used

- Naive Bayes Classifier
- Logistic Regression
- Support Vector Machine (SVM)

---
